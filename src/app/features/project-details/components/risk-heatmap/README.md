# RiskHeatmapComponent

Heatmap matrix for risk impact and probability before/after actions.

## Inputs
- `heatmap`: data for before and after grids.

## Behavior
- Renders two 3x4 grids with LOW/MEDIUM/HIGH/BLOCK columns.
- Applies tier background colors per column.

## Data-cy
- Root: `risk-heatmap`
- Before grid: `risk-heatmap-before`
- After grid: `risk-heatmap-after`
