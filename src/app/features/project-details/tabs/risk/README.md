# RiskTabComponent

Risk tab container for heatmap and risk card list.

## Responsibilities
- Fetches risk heatmap + entries with optional risk type filter.
- Renders risk type filter and add button.
- Displays skeleton while risks load.
- Drives confirm-then-refetch updates for risk fields.

## Data-cy
- Root: `risk-tab`
- Risk type filter: `risk-filter-type`
- Add button: `risk-add-button`
- Risk list: `risk-card-list`
- Empty state: `risk-empty-state`
