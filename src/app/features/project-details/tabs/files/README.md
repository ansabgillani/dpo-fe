# FilesTabComponent

Files tab container for file list management.

## Responsibilities
- Fetches files for the active project.
- Renders file rows with upload, download, and delete actions.
- Provides add button and empty state.
- Shows skeleton while loading.

## Data-cy
- Root: `files-tab`
- Add button: `files-add-button`
- List: `files-list`
- Empty state: `files-empty-state`
