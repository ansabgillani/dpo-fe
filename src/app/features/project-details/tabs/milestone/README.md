# MilestoneTabComponent

Milestone tab container for project deliverables.

## Responsibilities
- Loads milestone sets from filters API.
- Loads project milestones with optional milestone set filter.
- Renders milestone rows and empty/loading states.
- Updates start/end dates via service and refreshes list.

## Data-cy
- Root: `milestone-tab`
- Milestone set select: `milestone-set-select`
- Milestone list: `milestone-list`
- Empty state: `milestone-empty-state`
- Loading skeleton: `skeleton-milestone-list`
